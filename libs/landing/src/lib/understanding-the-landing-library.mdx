# Understanding the Landing Library

The `landing` library is a collection of components specifically designed for creating engaging and informative landing pages within the React Weapons of Choice (RWOC) monorepo. These components are styled with Tailwind CSS and Shadcn UI to ensure a consistent and cohesive design.

## Components

### 1. **`HeroSection`**
- **Description**: The `HeroSection` component is used to create a prominent introductory section on the landing page. It typically includes a title, subtitle, description, and a call-to-action button.
- **Usage**:
  ```tsx
  import { HeroSection } from '@rwoc/landing';

  const LandingPage = () => (
    <HeroSection
      title="Welcome to RWOC"
      subtitle="Discover the power of our monorepo"
      description="Streamline your development workflow with our comprehensive and customizable boilerplate."
      ctaPrimary={{
        text: 'Get Started',
        link: '/get-started',
      }}
      ctaSecondary={{
        text: 'Learn More',
        link: '/learn-more',
      }}
    />
  );
  ```

### 2. **`AboutSection`**
- **Description**: The `AboutSection` component provides an overview of the project or company, highlighting key information and values.
- **Usage**:
  ```tsx
  import { AboutSection } from '@rwoc/landing';

  const LandingPage = () => (
    <AboutSection
      title="About Us"
      description="Learn more about our mission and values."
    />
  );
  ```

### 3. **`CommunitySection`**
- **Description**: The `CommunitySection` component showcases the community around the project, including testimonials, user stories, and links to community resources.
- **Usage**:
  ```tsx
  import { CommunitySection } from '@rwoc/landing';

  const LandingPage = () => (
    <CommunitySection
      title="Join Our Community"
      description="Connect with other users and contribute to the project."
    />
  );
  ```

### 4. **`FeaturesSection`**
- **Description**: The `FeaturesSection` component displays a list of features or benefits of the project or product.
- **Usage**:
  ```tsx
  import { FeaturesSection } from '@rwoc/landing';

  const features = [
    { title: 'Feature 1', description: 'Description of feature 1' },
    { title: 'Feature 2', description: 'Description of feature 2' },
    { title: 'Feature 3', description: 'Description of feature 3' },
  ];

  const LandingPage = () => (
    <FeaturesSection
      title="Our Features"
      features={features}
    />
  );
  ```

### 5. **`PricingSection`**
- **Description**: The `PricingSection` component presents the pricing plans available for the product or service.
- **Usage**:
  ```tsx
  import { PricingSection } from '@rwoc/landing';

  const plans = [
    { name: 'Free', price: '$0', features: ['Feature 1', 'Feature 2'] },
    { name: 'Pro', price: '$49', features: ['Feature 1', 'Feature 2', 'Feature 3'] },
    { name: 'Enterprise', price: '$99', features: ['Feature 1', 'Feature 2', 'Feature 3', 'Feature 4'] },
  ];

  const LandingPage = () => (
    <PricingSection
      title="Pricing Plans"
      plans={plans}
    />
  );
  ```

## Conclusion

The `landing` library provides a set of components designed to create engaging and informative landing pages. By using these components, you can build a cohesive and visually appealing landing page that effectively communicates the value of your project or product.
