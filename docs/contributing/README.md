## Quick Start for Contributors

1. **Fork and Clone** - Follow the setup in the main README
2. **Review Code Standards** - Read the contributing guidelines
3. **Check Open Issues** - Find something to work on
4. **Create Feature Branch** - Use conventional branch naming
5. **Make Changes** - Follow project conventions
6. **Test Thoroughly** - Run `pnpm run test:all` and `pnpm run lint:all`
7. **Submit PR** - Use `@pr-description-helper` agent for description

## Contribution Areas

- **Documentation** - Improve guides, fix typos, add examples
- **Components** - Add shadcn/ui components to the library
- **Features** - Implement new capabilities
- **Bug Fixes** - Resolve reported issues
- **Testing** - Improve test coverage
- **Accessibility** - Enhance WCAG compliance
